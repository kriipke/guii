= Repository Structure

* `/dwm/` - source code of `iiwm` the `guii` version of `dwm`, a window manager
* `/dwm/misc` - miscellanous helper commands to be used with `guii`
* `/dwm/patches/` - this directory contains `dwm` patches applied
* `/st/` - source code of `iist` the `guii` version of `st`, a terminal
* `/st/patches/` - this directory contains `dwm` patches applied
* `/debian/` - `.deb` source
